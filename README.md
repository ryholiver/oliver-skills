# Oliver Skills

Oliver 的个人技能插件合集。

## 插件列表

- [any-file](any-file/) - 多模态文件处理工具集
- [feishu](feishu/) - 飞书生态全栈工具集
- [goals](goals/) - 目标与 OKR 管理工具集
- [interview](interview/) - 面试准备与求职工具集
- [learning](learning/) - 学习与知识管理工具集
- [obsidian](obsidian/) - Obsidian 知识库专用工具
- [others](others/) - 其他工具
- [pre-coding](pre-coding/) - 编码前准备工作
- [product](product/) - 产品工作流工具集
- [skill-tools](skill-tools/) - Skill 创建、优化、提取工具集
- [writing](writing/) - 写作与内容创作工具集

## 安装

```bash
/plugin install oliver-skills@ryholiver/oliver-skills
```
