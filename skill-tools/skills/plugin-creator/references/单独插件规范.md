# 单独插件规范

从 skill 打包成单个 plugin。

---

## 目录结构

```
my-plugin/
├── .claude-plugin/
│   └── plugin.json      # 必需
├── README.md            # 必需
├── skills/              # 必需，至少一个
│   └── my-skill/
│       └── SKILL.md
├── commands/             # 可选
│   └── my-command.md
├── agents/              # 可选
│   └── my-agent.md
└── hooks/               # 可选
    └── pre-commit.py
```

---

## plugin.json

```json
{
  "name": "my-plugin",
  "description": "插件描述",
  "version": "1.0.0",
  "author": { "name": "你的名字" },
  "license": "MIT"
}
```

**必需字段**：name、description、version

---

## README.md

```markdown
# My Plugin

简短描述。

## 安装

```
/plugin install my-plugin@claude-code-plugins-plus
```

## 使用方法

描述如何使用。

## 功能特点

- 功能 1
- 功能 2
```
