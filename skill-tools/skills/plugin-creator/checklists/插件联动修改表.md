# 插件联动修改表

> 改了 X，必须同步更新 Y。否则插件会出问题。

---

## 单插件

| 修改了什么 | 必须同步更新 |
|---|---|
| 重命名插件目录 | `plugin.json` 的 `name` 改为新目录名 |
| 修改 `plugin.json` 的 `name` | 插件目录名改为新 name |
| 修改 `plugin.json` 的 `description` | `marketplace.json` 中对应插件的 description |
| 修改 `plugin.json` 的 `version` | `marketplace.json` 中对应插件的 version |
| 移动插件目录位置 | `marketplace.json` 的 `source` 路径 |

## 多插件合集

| 修改了什么 | 必须同步更新 |
|---|---|
| 新增子插件 | `marketplace.json` 添加对应条目 + 根 `README.md` 添加插件链接 |
| 删除子插件 | `marketplace.json` 移除对应条目 + 根 `README.md` 移除插件链接 |
| 重命名子插件目录 | `marketplace.json` 更新 name 和 source |
| 修改子插件 `plugin.json` 的 name | 子插件目录名 + marketplace.json 的 name |
| 修改子插件 `plugin.json` 的 version | `marketplace.json` 中对应条目的 version |
| 移动子插件到其他分类 | `marketplace.json` 更新对应条目的 category |
