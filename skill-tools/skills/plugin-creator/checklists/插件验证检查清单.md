# 插件验证检查清单

> 🔴 必须通过 → 不通过会影响安装使用
> 🟡 建议补充 → 不影响使用但建议补齐

***

## 🔴 必须通过

- [ ] `.claude-plugin/plugin.json` 存在
- [ ] `name` 存在、kebab-case、与插件目录名一致
- [ ] `skills/` 下至少有一个 skill

## 🟡 建议补充

- [ ] `description` 不为空
- [ ] `version` 符合语义化版本
- [ ] `author` 结构正确
- [ ] `license` 使用标准标识符
- [ ] `README.md` 存在且有内容
- [ ] 所有目录名 kebab-case
- [ ] 没有多余文件（`__pycache__`、`.DS_Store`、`.git` 等）
