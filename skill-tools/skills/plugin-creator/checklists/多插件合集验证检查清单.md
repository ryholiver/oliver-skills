# 多插件合集验证检查清单

> 🔴 必须通过 → 不通过会影响安装使用
> 🟡 建议补充 → 不影响使用但建议补齐

***

## 🔴 必须通过

- [ ] 每个子插件有自己的 `plugin.json`
- [ ] 每个子插件 `name` 与目录名一致
- [ ] `marketplace/marketplace.json` 存在
- [ ] 所有已登记的插件目录真实存在
- [ ] `marketplace.json` 没有残留已删除插件的条目
- [ ] `marketplace.json` 没有遗漏新增插件的条目
- [ ] 每个子插件 `plugin.json` 的 name/version/description 与 marketplace 对应条目一致

## 🟡 建议补充

- [ ] `marketplace.json` 的 `name` 与合集目录名一致
- [ ] 每个插件的 `category` 使用预设分类
- [ ] 根 `README.md` 包含插件列表
- [ ] 所有目录名 kebab-case
- [ ] 没有多余文件
