# 插件排障清单

> 插件装不上、用不了时查。

---

| 现象 | 原因 | 修复 |
|---|---|---|
| "Invalid plugin" | plugin.json 缺少必需字段或 JSON 格式错误 | 检查 name/description/version |
| "Plugin not found" | 路径不存在或目录名有误 | 确认目录存在 |
| 装完后 /plugin list 看不到 | marketplace.json 没注册 | 检查市场登记 |
| 安装成功但文件缺失 | 目录结构不完整 | 对照验证清单检查骨架 |
| macOS/Linux 上不工作 | 路径大小写不匹配 | 全部改为小写 + 连字符 |
