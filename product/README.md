# Product

产品工作流工具集。

## 安装

```
/plugin install product@claude-code-plugins-plus
```

## 技能

- ai-agent-prd-writer - AI Agent PRD 编写
- ai-product-teardown - AI 产品拆解
- product-insight-miner - 产品洞察挖掘
- startup-ideation - 创业构思
- teardown-doc-writer - 拆解文档编写
- vibe-coding - Vibe Coding
- vibe-prd-writer - Vibe PRD 编写
