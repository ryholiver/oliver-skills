# mmx-cli

MiniMax AI 多模态生成 CLI：通过 `mmx` 在终端生成文本、图片、视频、语音、音乐，并执行 web 搜索、管理 MiniMax API 资源。

## 安装

```
/plugin install mmx-cli@claude-code-plugins-plus
```

## 技能

- mmx-cli — MiniMax AI 多模态生成 CLI 使用指南

## 前置依赖

```bash
npm install -g mmx-cli
mmx auth login --api-key sk-xxxxx
```
